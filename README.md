### Usage

### Controls :video_game:

*Alternate controls apart from the UI*

| Key | Purpose|
|-----|--------|
| `s` | Save the image as PNG |
| `g` | Generate a new image |
| `n` | Switch to next pattern generator |
| `]` | More pattern blocks |
| `[` | Less pattern blocks |
| `+` | Make the pattern more complex |
| `-` | Make the pattern less complex |
